# IPinfo
The Trusted Source for IP Address Data.
With IPinfo, you can pinpoint your users’ locations, customize their experiences, prevent fraud, ensure compliance, and so much more. IPinfo is a free IP data API provider specializing in IP geolocation, though also offering endpoints for ASN, IP to company, IP to carrier, IP ranges, and hosted domains. The geolocation API returns a JSON response containing data fields describing the specified IP&#x27;s region, country, city, postal code, coordinates, and more. IPinfo APIs handle over 500 million requests per day and have been online since 2013. It’s reliable, scalable, simple, and straightforward to use!.
